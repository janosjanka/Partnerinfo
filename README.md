# Partnerinfo
Partnerinfo is an Online Marketing System with modern SPA-based CMS services (client-side routing, JavaScript modules, blazing fast, modern web designer, ...). In contrast to most of online marketing systems, it does not contain any built-in business functions, instead, everybody itself can create business activities and use those without extra efforts.

## Design Goals

- Project Management
- Workflow Engine for Web Actions
- Workflow Designer
- Action link editor
- Contact management
- Newsletter management
- Business tagging system
- Unlimited web page, domain management / project
- Blazing fast web designer (HTML, CSS, JS editors)

- Logging & Rule-based filters
